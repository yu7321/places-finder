Places Finder

A configurable pipeline for finding, deduplicating, and mapping places via the Google Places API.

Open live demo map →

What it does

Point it at a location and a category (agriturismi, wineries, specialty coffee, anything else Google Places indexes). It runs multiple Places queries to improve recall, optionally scrapes a complementary source, merges everything with coordinate-based deduplication, and renders the result as a self-contained interactive HTML map.

On the demo map

The live demo shows 15 synthetic places around Chianti, Tuscany, demonstrating all three marker types:

  • Leaf — found via Google Places only
  • Home — found via agriturismo.it only
  • Star — found by both sources (most trustworthy)

Marker color encodes rating — green for 4.7+, blue for 4.4–4.6, orange for below 4.4. Click any marker to see the popup with address, phone, license codes, and source attribution.

Try it yourself

Clone the repo, install dependencies, paste a Google Places API key into config.yaml, and run:

  • python main.py --location "Chianti, Italy" --radius 20 — discover places
  • python merge.py "*.csv" --output merged.csv — dedup across sources
  • python map.py merged.csv — render your own HTML map
